# Enrichment Guidelines

Use this reference when creating `outputs/ipa-examples.csv`.

## Field Rules

### 法语单词

- Preserve the order from `outputs/duolingo-words.csv`.
- Keep accents and spelling correct.
- If the screenshot spelling appears wrong or uncertain, explain before changing it.

### 法语英标

- Use French IPA in square brackets, for example `[plɛ]`, `[ku.ʁjɛl]`, `[bwa.sɔ̃ fʁɛʃ]`.
- For phrases, show natural pronunciation across the phrase when confident.

### 例句

- Write one natural, short French sentence that uses the target word directly.
- Put the English sentence on the next line in the same CSV cell.
- Use this shape:

```text
✅French sentence.
-- English sentence.
```

- Avoid meta-sentences like `J'utilise "word" dans une phrase.`

### 词性/用法

- Start with `⭐`.
- Explain in beginner-friendly Chinese.
- Prefer clear Chinese terms over unexplained grammar labels:
  - `动词原形`
  - `直陈式现在时`
  - `条件式现在时`
  - `名词 阳性/阴性`
  - `形容词 阴阳同形`
  - `固定搭配`
- For conjugated verbs, name the infinitive, tense/mood, and person.

### 英文翻译

- Keep it short, ideally 1-3 English words.
- Use the core learning meaning, not every Duolingo variant.

### 中文翻译

- Use Chinese only.
- Keep it concise and useful for memorization.
